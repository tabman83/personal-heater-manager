# personal-heating-manager
A personal heating manager system with Arduino and Raspberry Pi.

## Prerequisites

Install the Bower package manager:
```
$ npm install -g bower
```

Install the Gulp task manager:
```
$ npm install gulp -g
```

## Getting up and running
```
$ npm install
$ npm start
```
